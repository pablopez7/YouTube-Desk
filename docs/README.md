# Screenshots

Place your app screenshots here.

## Recommended screenshots:

1. `screenshot.png` - Main app view with tabs
2. `context-menu.png` - Right-click context menu in action
3. `dark-theme.png` - Full app showing dark theme

## Taking screenshots:

1. Run the app: `npm run start`
2. Navigate to YouTube
3. Use Windows Snipping Tool or `Win+Shift+S`
4. Save as `screenshot.png` in this folder
